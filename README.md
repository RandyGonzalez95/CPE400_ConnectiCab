## Makefile Instructions
The makefile works as expected and must be updated with new files added in.

```bash
mkdir build
cd build
cp ../makefile .
make
./ConnectiCab
```
Alternatively you can run the bash script provided "runProgram.sh" by simply running ./runProgram.sh
